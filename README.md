# Get Started

- Open your favorite Terminal
## Installation
```sh
git clone https://github.com/reeder32/Bakery.Solution.git
```
#### Run Tests
```sh
cd BakeryTests
dotnet test
```
#### Run Program
```sh
cd Bakery
dotnet run
```
